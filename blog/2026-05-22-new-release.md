We’re excited to announce the release of **CosmWasm 3.0.7**.
This minor patch release includes contributions from our community. 
You can find the full list of changes in the
[CHANGELOG](https://github.com/CosmWasm/cosmwasm/blob/v3.0.7/CHANGELOG.md).

This release ships updated versions of **cosmwasm-std**, **wasmvm**, and **wasmd**
in supported `0.70.x` and `0.61.x` series.
